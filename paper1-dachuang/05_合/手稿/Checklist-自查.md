# 论文自查 Checklist

> 基于《科研论》工具1.8 + 目标期刊（Food Chemistry / CEA）要求定制
> 投稿前逐条核对，全部勾选后发送

---

## 引言 (Introduction)

- [ ] 是否清晰引出研究的科学问题？（羊肚菌干燥过程风味品质动态变化）
- [ ] 是否交代了研究大背景？（食用菌干燥产业 + 风味品质控制需求）
- [ ] 是否综述了现有研究？（至少引用 A组7篇 + C组3篇综述）
- [ ] **是否引用了 Horticulturae 2024 并明确差异化？**（必须）
- [ ] 是否引用了 Nat Commun 2024 啤酒风味 ML 作为方法论标杆？
- [ ] 是否明确阐述本研究的 Contribution？（5维差异化：动态时序/三模态/DL/SHAP/因果通路）
- [ ] 逻辑链：产业需求→现有局限→本文创新→研究目标，逐段收缩

---

## 方法 (Methods / Materials and Methods)

- [ ] 羊肚菌来源、品种、采收时间是否说明？
- [ ] 干燥设备型号、参数范围是否完整？（温度/风速/装载量）
- [ ] 采样时间点和重复数是否明确？（6-8点 × 3重复）
- [ ] HS-SPME-GC-MS 条件完整？（萃取头/平衡时间/升温程序/柱型号）
- [ ] LC-MS 条件完整？（色谱柱/流动相/质谱参数/扫描范围）
- [ ] 数据预处理流程可复现？（基线校正/峰对齐/归一化方法）
- [ ] OAV 计算公式 + 嗅觉阈值来源引用
- [ ] ML 模型超参数是否明确？（RF: n_estimators/max_depth; XGBoost: lr/max_depth; LSTM: layers/units）
- [ ] 评估指标合理且说明选择理由？（R² / RMSE / MAE → 回归任务标准）
- [ ] 是否说明数据划分策略？（如 70/15/15 或 k-fold CV）
- [ ] 是否说明随机种子固定（seed=42）以确保可复现？

---

## 结果与讨论 (Results and Discussion)

- [ ] 每个 Figure 都有对应的文字描述和讨论？
- [ ] Fig2（干燥动力学+挥发物时序）：是否分析了水分迁移与挥发物释放的耦合？
- [ ] Fig3（多模态差异）：是否报告了 PCA/PLS-DA 的 R²X/R²Y/Q²？
- [ ] Fig4（ML性能）：是否对比了所有模型（LR/RF/XGBoost/LSTM）？是否给出训练/验证/测试集结果？
- [ ] Fig5（SHAP）：是否解释了 top-5 特征的生物学/化学含义？不只是技术描述
- [ ] Fig6（通路模型）：是否 5 条通路均有文献支撑？KEGG ID 是否标注？
- [ ] 是否与 Horticulturae 2024 的结果做了定量对比？（如"我们的 RMSE 比 RF 降低 X%"）
- [ ] 是否与牛肝菌风味预测 2025 做了方法对比？
- [ ] 是否讨论了与 Nat Commun 2024 啤酒 ML 的方法论差异？
- [ ] 机制解释是否有代谢组学数据支撑？（不是纯推测）
- [ ] 是否有"意料之外"的结果但未解释？（如某个特征 SHAP 值为负但预期为正）
- [ ] 是否避免了过度声称？（如"首次"→需核实，"显著"→需 p 值）

---

## 结论 (Conclusion)

- [ ] 是否重申解决了什么问题？（干燥条件→风味动态 AI 预测）
- [ ] 是否给出了最关键的 2-3 个数值发现？（如"XGBoost R²=X.XX, 关键通路XX条"）
- [ ] 是否明确指出局限性？（如"样本量有限/单一品种/缺乏外推验证"）
- [ ] 是否提出了未来方向？（如 G07 Digital Twin / 跨品种迁移）
- [ ] 是否与引言中的研究目标呼应？

## 摘要 (Abstract)

- [ ] 包含：背景1句 + 空白1句 + 方法2-3句 + 关键结果2-3句 + 结论/意义1句
- [ ] 字数符合目标期刊？（150-250词，视期刊而定）
- [ ] 是否包含关键数字（R²/RMSE/化合物数/通路数）？
- [ ] 关键词 5-8 个（Morchella, drying, flavor prediction, machine learning, SHAP, metabolomics, volatilomics）

## 格式与图表

- [ ] 图表清晰度达标（≥600 DPI for Food Chemistry）
- [ ] 图表格式正确（TIFF/EPS/高分辨率 JPEG，RGB模式）
- [ ] 坐标轴标签完整、单位正确
- [ ] 色盲友好配色（避免红绿单独使用）
- [ ] 图例位置不遮挡数据
- [ ] 参考文献格式符合目标期刊（Food Chemistry: numbered, 方括号）
- [ ] 缩写首次出现有完整定义（如 HS-SPME-GC-MS, OAV, SHAP, Deff）
- [ ] 所有数据有误差棒/标准偏差标注
- [ ] 统计显著性标注（*, **, *** 或 p 值）

## 语言与逻辑

- [ ] 段落间有清晰的逻辑过渡（避免"积木式"堆砌）
- [ ] 不存在 >40 词的长句（或每段最多1句）
- [ ] 专业术语使用一致（全篇统一用"volatilomics"而非混用"volatile analysis"）
- [ ] 时态正确（Methods 用过去时，Results 用过去时，Discussion 用现在时）
- [ ] 无中式英语（如 "obvious difference" → "significant difference"）

## 常见拒稿原因（提前预防）

| 问题 | 预防措施 |
|------|---------|
| "增量贡献不足" | 5维差异化表放在 Introduction 末尾 |
| "方法不够新" | 强调 DL + SHAP + 三模态时序整合 |
| "缺乏独立验证集" | 保留 15-20% 数据做 hold-out test set |
| "机制解释肤浅" | Fig6 必须回溯到 KEGG 通路 + 文献 |
| "写作质量差" | 投稿前至少 2 轮英文润色 |
| "对标文献遗漏" | 文献矩阵 31 篇全部检查引用完整性 |
