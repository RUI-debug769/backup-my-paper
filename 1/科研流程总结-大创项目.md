## 一、项目总览

本大创项目以羊肚菌（Morchella spp.）为研究对象，构建**整合干燥动力学、HS-SPME-GC-MS 挥发组学与 LC-MS 代谢组学的动态时间序列可解释 AI 框架**，实现从"静态描述"到"因果机制驱动的风味演化预测"的范式升级。

**核心科学问题**：干燥如何调控代谢转化与挥发性风味演化？AI 如何预测风味品质动态？

**一句话创新总结**（来自机会清单 G01+G02+G04）：
> "首次构建整合干燥动力学、HS-SPME-GC-MS 挥发组与 LC-MS 代谢组的**动态时间序列**可解释 AI 框架，从静态描述升级为**因果机制驱动的风味演化预测**，揭示干燥诱导的 5 条代谢通路如何时序调控羊肚菌风味形成。"

### 差异化重新论证

Horticulturae 2024 已实现 GC-IMS+LC-MS+RF 三模态联合，但**仅做静态终点比较**。本项目的核心创新从"首次三模态联用"调整为：

> **"首次动态时序多模态 + 可解释 AI 因果机制 + 干燥动力学驱动风味预测"**

5 个维度全部超越：时间序列（全新）、多模态深度（三模态vs二模态+RF）、AI方法（DL+SHAP vs RF）、机制深度（5通路因果模型 vs 无）、预测能力（有 vs 无）。

### MVE 方案（分层设计）

```
Tier-1（最小可行，6-8周）：
  干燥动力学 + LC-MS代谢组 + AI（RF/XGBoost）
  → 预测关键品质动态
  → 成本最低，可单独成文

Tier-2（完整版，追加4-6周）：
  + HS-SPME-GC-MS挥发物 + OAV风味标签
  → LSTM/Transformer时序预测 + SHAP因果解释
  → 打通"前体代谢物→挥发物"因果链
```

### 三问校验（Idea-01）

1. **真实世界锚点**：羊肚菌干燥加工企业需要预测"什么时候风味最好"，目前完全依赖经验
2. **可被实验检验**：GC-MS 检测时序挥发物变化 → OAV 区分风味好坏 → 干燥参数与风味关联强度可量化
3. **第一支点**：先用文献数据模拟验证"干燥动力学参数→风味"的关联强度

## 九、经验沉淀

### 方法论
1. **检索式必须英文**：中文关键词在 PubMed/WoS 命中率约等于零
2. **Zotero 全库筛选是高效策略**：比逐库检索节省 ~70% 时间，3340篇→31篇只需 Python 关键词 + 人工校验
3. **发现对标论文不可怕，可怕的是不知道它存在**：Horticulturae 2024 的发现虽然让"首次三模态联用"不成立，但迫使差异化更精准
4. **五步法不缩水，但数字可÷10**：3篇全文精读 + 1张手绘框架图的深度 > 100篇标题浏览

### 工具链
- **Zotero Better BibTeX** → CSV 导出 → Python 关键词筛选 → 文献矩阵自动化
- **keyanlun 五步法** 提供清晰的阶段门控，防止在【搜】阶段无限延伸
- **14 Agent 系统** 虽然是模拟角色，但定义了清晰的交接规范和产出格式

### 待改进
- WoS/Scopus 中文检索尚未执行（引言中可能需要少量中文文献背景）
- 缺少独立的 Web of Science 检索（PubMed 和 WoS 覆盖有重叠但不完全）
- 目前尚未进入实验阶段，所有假设和设计均为纸面方案，需要通过 MVE 验证
