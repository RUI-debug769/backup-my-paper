## 三问校验（每个 idea 启动前必答）

### Idea-01
1. **真实世界锚点是什么？** → 羊肚菌干燥加工企业需要知道"什么时候风味最好"来优化工艺参数，目前完全依赖经验
2. **逻辑正确是否等于真实？** → 需要实验数据验证：(a) GC-MS 能否检测到干燥过程中的挥发物时序变化？(b) OAV 能否有效区分"好风味"和"差风味"？(c) 干燥动力学参数是否与风味变化强相关？
3. **第一个支点/测试项是什么？** → 先用文献数据（已发表的干燥曲线 + 挥发物数据）做可行性模拟，验证"干燥动力学参数 → 风味"的关联强度

### Idea-02
1. **真实世界锚点是什么？** → 传统感官评定需要训练有素的评定员（10-15人），成本高、主观性强；OAV 是化学标准，可客观量化
2. **逻辑正确是否等于真实？** → OAV>1 = 有感官贡献，但OAV总和≠感官体验。需要与至少小规模感官评定做相关性验证
3. **第一个支点/测试项是什么？** → 从已发表文献中提取羊肚菌关键挥发物的嗅觉阈值 → 计算 OAV → 与文献中的感官描述做定性对比

## MVE 方案概要（分层设计）

### Tier-1：最小可行（先立住一篇）
```
目标：验证"干燥动力学 + LC-MS + AI → 预测关键品质动态"可行
数据：72-108组干燥实验（3温度 × 3-4时间点 × 3重复）
输入：干燥动力学参数（Deff, 活化能, 水分比, aw）
输出：LC-MS 差异代谢物丰度 + 理化品质指标
模型：LR → RF → XGBoost 基线
评估：R² > 0.85, RMSE < 15%
时限：6-8周
成本：最低（仅需 LC-MS 上样）
```

### Tier-2：完整版（机制闭环）
```
追加：HS-SPME-GC-MS 挥发物 + OAV 风味标签
打通：前体代谢物（LC-MS）→ 挥发物（GC-MS）因果链
模型：LSTM / Transformer 时序预测
评估：R² > 0.90, RMSE < 10%, SHAP 一致性 > 80%
时限：追加 4-6周
```
