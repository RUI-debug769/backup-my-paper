## 二、实验设计

```
物种：羊肚菌 Morchella sextelata（六妹羊肚菌）
来源：四川成都阳马镇农场
样品：菌盖长度 9-11 cm，无病虫害，形态饱满

干燥条件（热风干燥）：
  LT: 45°C → 11.5 h 完成
  MT: 55°C → 11.0 h 完成
  HT: 65°C → 9.0 h 完成

每组 5 个生物学重复，共 15 份样品（每份 1kg 鲜菇）
```

### ⚠️ 关键限制

**只有干燥终点数据，无时间序列采样！** 这意味着：
- 论文比较的是"不同温度干燥后的成品"，而非"干燥过程中风味如何演化"
- 这正是我们项目要超越的核心局限
- 对于 MVE：只能做"温度→风味"的静态预测，无法做时序预测

## 四、核心结果

### 4.1 干燥曲线 (Figure 1)

```
可用数据：MR (水分比) vs 时间曲线
  HT: 0h(1.0) → 1.5h(~0.82) → 3h(~0.65) → 4.5h(~0.48) → 6h(~0.30) → 7.5h(~0.14) → 9h(~0.02)
  MT: 0h(1.0) → ... → 11h 完成
  LT: 0h(1.0) → ... → 11.5h 完成
```

**可提取数据：** Figure 1 干燥曲线 → WebPlotDigitizer 数字化 → MR 时间序列（~7个点/温度）

### 4.2 挥发物指纹 (Figures 2-4)

```
48 种挥发物检出，33 种鉴定，15 种未知
分类变化趋势（随温度升高）：
  ↑ 醛类 (aldehydes)       — 脂肪氧化 + Maillard
  ↑ 烃类 (hydrocarbons)     — Maillard 反应
  ↑ 吡嗪类 (pyrazines)      — 坚果/烘烤风味（Maillard）
  ↓ 酸类 (acids)           — 乙酸显著下降（不愉快酸味减少）
  ↓ 醇类 (alcohols)        — 酯化/氧化消耗
  ↓ 酯类 (esters)          — 高温分解
  — 酮类 (ketones)         — 无显著变化
```

**RF 指示性化合物（Figure 7a）：**
| 温度 | 指示化合物 | 风味描述 |
|------|-----------|---------|
| LT (45°C) | 1-己醇 (1-hexanol) | 油脂、酒精味 |
| | 3-甲基丁酸乙酯 (ethyl 3-methylbutanoate) | 果香 |
| | 丁醛-D (butanal-D) | 麦芽、巧克力 |
| | 3-甲基丁-1-醇-M | — |
| | 2-甲基-1-丙醇 (2-methyl-1-propanol) | 醚、酒味 |
| MT (55°C) | 己醛-D (hexanal-D) | 青草味（羊肚菌主要风味） |
| | 己醛-M (hexanal-M) | 青草味 |
| | 戊醛-D (valeraldehyde-D) | 果香、坚果 |
| HT (65°C) | 戊醛-M (valeraldehyde-M) | 果香、坚果 |
| | + 5 个未知化合物 | — |

### 4.3 非挥发性代谢物 (Figures 5-7)

```
1645 种代谢物鉴定（正+负离子模式，Table S1）
9 种差异表达模式聚类
KEGG 显著富集通路（p<0.05）：
  1. 果糖和甘露糖代谢 (ko00051)
  2. 甘油酯代谢 (ko00561)
  3. N-聚糖生物合成 (ko00510)

氨基酸 + 核苷酸 → 随温度升高显著增加 (p<0.05)
  5'-核苷酸 + 风味氨基酸 → 同趋势
有机酸 → 趋势不显著 (p>0.05)
```

| 温度 | 指示代谢物 | 意义 |
|------|-----------|------|
| LT | 山梨醇-6-磷酸 (Sorbitol 6-phosphate) | 甜味剂合成调控 |
| | 高车前苷 (Homoplantaginin) | — |
| MT | 组氨酸 (Histidine M156T368) | 风味氨基酸 |
| HT | Glu-His, 4-羟基-L-谷氨酸, Ala-Phe, Arg-Trp-Arg, 2-丙烯酰胺衍生物 | 氨基酸+黑色素前体（⚠️ 高温风险） |

### ⚠️ 数据局限性（对 MVE 的影响）

| 局限 | 影响 | 缓解方案 |
|------|------|---------|
| **无时间序列** | 无法直接验证时序预测假设 | 需结合其他有时间序列的论文；或改为"温度→风味"静态预测验证 |
| **仅 3 个温度点** | 训练数据极少 (n=3 或 n=15 含重复) | LOOCV; Bootstrap; 降低模型复杂度 |
| **挥发物为相对丰度** | 无法计算 OAV | 用归一化丰度替代；OAV 验证留到 Tier-1 |
| **GC-IMS 非 GC-MS** | 鉴定能力有限（33/48已鉴定） | 仅使用已鉴定化合物 |
| **无生物学重复的个体数据** | 只能使用均值 ± SD | 若补充材料有个体数据则更优 |

## 六、结论摘要

**论文核心发现：**
1. 温度↑ → 干燥效率↑ → 醛类/烃类/吡嗪类↑, 酸类/醇类/酯类↓
2. 低温 (45°C) 保留醇酯类（果香/酒香），中温 (55°C) 保留己醛（青草味=羊肚菌特征风味），高温 (65°C) 产生更多吡嗪但增加黑色素风险
3. RF 模型成功筛选出各温度的指示化合物和代谢物
4. 氨基酸和核苷酸随温度升高而增加（鲜味增强）

**对我们项目的启示：**
1. ✅ 确认了"温度影响风味"的基本关联——这支持我们的核心假设
2. ✅ 提供了 RF 基线模型——我们可以直接用 XGBoost/SHAP 超越它
3. ✅ 确认了关键风味化合物类别——为我们选择目标化合物提供依据
4. ❌ **无时间序列**——这既是我们项目的差异化优势，也意味着这篇论文不能单独用于时序验证
5. ❌ 论文本身就是我们说的"静态终点比较"——我们的创新定位是正确的
