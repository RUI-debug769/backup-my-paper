### B-01 详情：原始数据来源

```
问题：MVE Phase 1 需要干燥动力学 + LC-MS 数据来训练基线模型。
     数据从哪里来？

选项 A：已有实验数据（前期积累 or 导师提供）
  → 优势：立即可启动 MVE
  → 风险：数据质量/完整性未知

选项 B：需从头采集
  → 优势：数据质量可控，实验设计自由度大
  → 风险：启动延迟 1-2 个月，需协调设备

选项 C：先用文献数据做概念验证（已有 Guo L 2024 等论文数据可参考）
  → 优势：无需等待实验，可立即训练基线模型
  → 风险：文献数据粒度可能不够，最终仍需实测
```

### B-02 详情：GC-MS 机时

```
问题：HS-SPME-GC-MS 是挥发物检测的核心平台。
     预约周期多长？一次能跑多少样？

需确认：
- 设备位置（本院 or 外院 or 外校？）
- 预约流程与周期（提前多久？是否有排队？）
- 单次可测样品数（一个干燥批次 6-8 个时间点 × 3 重复 = 18-24 样）
- 费用（若有）
- 数据分析支持（是否有现成的 NIST 谱库？）
```

### B-03 详情：LC-MS 外送

```
问题：非靶向代谢组学是 Tier-1 核心数据源。
     外送费用和周期如何？

需确认：
- 外送机构（哪个大学/公司？）
- 单价（~500-800 元/样是估算，实际？）
- 最低样本数
- 周期（从送样到拿到数据）
- 数据分析包含什么（峰对齐/鉴定/定量/差异分析？）
- Tier-1 最少需要多少样？（Tier-1 若 3 个干燥条件 × 5 时间点 × 3 重复 = 45 样 → ~2.2-3.6 万元）
```

### B-04 详情：计算资源

```
问题：深度学习模型（LSTM/Transformer）训练需要 GPU。
     本地有可用 GPU 吗？

需确认：
- 本地有无 GPU？型号？（如 GTX 1060 / RTX 3060 / 无）
- 若无，是否可用学校服务器 or 云计算（AutoDL/Colab/恒源云）？
- 云计算预算？
- 替代方案：先用 RF/XGBoost（CPU 可跑），深度学习放 Tier-2
```
