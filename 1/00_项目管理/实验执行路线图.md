### 阶段 B: 预实验 (1 周)

```
做什么:
  只跑 55°C 一个温度, 3 个时间点 (0, 3h, 终点)

  Day 1: 切菇 → 干燥 → 采样 (3 个时间点 × 3 重复 = 9 样)
  Day 2: GC-MS 跑 9 个样 (每个 ~35 min)
  Day 3: LC-MS 跑 9 个样
  Day 4: 检查数据质量:
    □ GC-MS 是否检出 1-辛烯-3-醇、己醛、苯甲醛? (Liu 2024 确认有)
    □ LC-MS 峰数量是否在 1500+ 范围? (Liu 2024: 1645 种)
    □ MR 曲线是否平滑下降?
    □ 色谱峰形是否对称?
  Day 5: 如有问题, 调整参数后重跑

为什么:
  全流程验证, 发现并解决所有操作问题
  避免正式实验跑了一半发现某个环节卡住

怎么做:
  1. 洗菇 → 去柄 → 切 2mm 片
  2. 平铺烘箱托盘, 55°C
  3. t=0: 取 3 份 → 称重 → 拍照 → 液氮速冻 → -80°C
  4. t=3h: 同上
  5. 终点 (MC<15%): 同上
  6. 所有样送 GC-MS + LC-MS
```

### 阶段 C: 正式实验 (4-6 周)

```
做什么:
  四个温度, 每个温度跑 1 周

  Week 1: 45°C (终点约 11.5h)
    每天跑 1 批 (R1/R2/R3, 共 3 批)
    每批在 6 个时间点各取 1 份样
    每份样分 3 管: GC-MS用 / LC-MS用 / 备用

  Week 2: 55°C (终点约 11h)
    同上

  Week 3: 65°C (终点约 9h)
    同上

  Week 4: 75°C (终点约 7h)
    同上

为什么:
  按温度从低到高, 干燥时间逐渐缩短
  每个温度独立完成, 避免混淆

怎么做:
  每个采样点的操作 SOP:
    1. 从烘箱取出 1 份样品
    2. 称重 → 记录 → 计算 MR
    3. 拍照 (手机即可, 固定距离+光源)
    4. 液氮速冻 30s
    5. 分装到标记好的冻存管
    6. 记录: "T55-R2-T3 已采样, MR=0.38"
    7. 冻存管立刻放入 -80°C
```

### 阶段 D: 仪器分析 (与阶段 C 交错进行)

```
做什么:
  每攒够一批样品就送 GC-MS 和 LC-MS

  GC-MS 流程 (每个样品 ~40 min):
    1. 冻干样品研磨成粉
    2. 称取 0.1g 到顶空瓶
    3. 加 5mL 水 + 10μL 内标
    4. 55°C 平衡 15min → SPME 萃取 30min
    5. 250°C 解吸 5min → GC-MS 35min

  LC-MS 流程 (每个样品 ~15 min):
    1. 称取 100mg 冻干粉
    2. 加 1mL 甲醇/乙腈/水 (2:2:1)
    3. 离心 14000g × 20min → 取上清
    4. 真空离心干燥 → 100μL 复溶
    5. UHPLC-QTOF 分析

为什么:
  分批送样避免仪器独占
  Methods 文档里有所有参数, 直接给仪器管理员

怎么做:
  提前 1 天预约机时
  每次送 12-18 个样 (一个温度一个重复)
  每 10 个样插 1 个 QC 样 (混合所有样品)
```

```
做什么:
  H1. Results 第一稿 (Week 1)
     按 Figure 1→7 的顺序写
     只写"数据说了什么", 不解释"为什么"

  H2. Discussion 第一稿 (Week 2)
     解释"为什么"
     与 Liu 2024 / Liao 2025 / Guo 2025 比较
     Gap statement 已写好, 直接引用

  H3. Introduction (Week 2)
     从 Gap statement 往前写
     背景 → 已有研究 → 空白 → 本研究

  H4. Abstract + Conclusions (Week 3)

  H5. 内审 → 修改 → 润色 (Week 3-4)
     先给导师看
     再给同实验室同学看
     最后用语法工具润色

为什么:
  手稿是最终交付物, 需要反复打磨
  Methods 已经写好, 省了 30% 工作量

怎么做:
  每写完一节, 朗读一遍, 检查逻辑流
  Introduction 的最后一段 = gap_statement_introduction.md
```
