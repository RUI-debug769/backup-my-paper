# 论文自查 Checklist（基于《科研论》工具1.8 行动3）

## 引言 (Introduction)
- [ ] 是否清晰引出研究的科学问题/技术瓶颈？
- [ ] 是否交代了研究大背景（为什么重要）？
- [ ] 是否综述了现有研究、指出不足（gap）？
- [ ] 是否明确阐述本研究的 Contribution？
- [ ] 逻辑链是否从大到小自然过渡到本文工作？

## 方法 (Methods / Materials and Methods)
- [ ] 关键步骤描述是否足够详细以供复现？
- [ ] 材料/试剂/数据来源是否完整？
- [ ] 参数范围/超参数是否明确？
- [ ] 评估指标是否合理且说明选择理由？

## 结果与讨论 (Results and Discussion)
- [ ] 每个图表是否都有对应的文字描述和讨论？
- [ ] 是否围绕核心问题展开论证，而非堆砌数据？
- [ ] 是否与文献对比，说明自己的独特性？
- [ ] 机理/解释是否有证据支撑（不是纯推测）？
- [ ] 是否有"意料之外"的结果但未解释？

## 结论 (Conclusion)
- [ ] 是否重申解决了什么问题？
- [ ] 是否明确指出局限性和未来方向？
- [ ] 是否与引言中的研究目标呼应？

## 摘要 (Abstract)
- [ ] 是否包含：背景1句 + 问题1句 + 方法1-2句 + 关键结果2-3句 + 结论/意义1句？
- [ ] 字数是否符合目标期刊要求？

## 格式与图表
- [ ] 图表清晰度达标（≥300 dpi）？
- [ ] 坐标轴标签完整、单位正确？
- [ ] 参考文献格式符合目标期刊要求？
- [ ] 缩写首次出现是否有完整定义？
- [ ] 所有数据是否有误差棒/统计检验？

## 语言与逻辑
- [ ] 段落间是否有清晰的逻辑过渡？
- [ ] 是否存在过长的句子（>40词）？
- [ ] 专业术语使用是否一致？

## 投稿准备
- [ ] Cover Letter 是否突出创新性和匹配度？
- [ ] 所有作者是否已确认作者顺序和版本？
- [ ] SI 补充材料是否完整？
